# OpenOps
0.1.1 - first commit.
At a high level - What is this all about?
As a NOC engineer, I see systems crash everyday for any reasson.
When it happens, money is leaking. So idealy, there isn't much time to think.
Thats why I believe making a human-like capable reasoning machine on top of complex systems can be huge.

My philosophy is that the reasoning mechanism will have all of the data the NOC has, with all of the troubleshooting tools it has. If it has less than that, it defeats the point.
Only after it does, there will be the complex agentic reasoning mechnism.

My long term vision is creating a technology where large production systems would stop being vague, and actually utilize AI to troubleshoot and maintain it, which in my view its very possible but not yet done.

# Same-Day Non-Goals

The following are explicitly out of scope for the first implementation.

## Interfaces

- Web interface
- REST or gRPC API

## Infrastructure

- Database
- Persistent memory
- Cloud deployment
- Authentication

## Integrations

- Prometheus
- Grafana
- Elasticsearch
- External plugins
- Third-party services

## Runtime

- Multi-agent workflows
- Autonomous remediation
- Write actions against Kubernetes
- Long-running background processes

## Product Scope

- Additional incident scenarios
- Generic AI assistant capabilities
- Production-scale performance
